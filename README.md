# Requirements

[Nodejs](https://nodejs.org/en)

[Piwigo](https://piwigo.org)

[Automatic webUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui)

# instructions

1. fill in the details in Config.js 
2. Run `npm install`
3. run `node AutoTagger.js`
